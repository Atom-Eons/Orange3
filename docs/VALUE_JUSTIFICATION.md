## Why Local-First Matters

Comparable tools often require:

- sending project state to someone else's servers
- trusting opaque retention policies
- paying per seat per month
- losing access if the service changes or shuts down

ORANGEBOX is local-first. The data is on your machine. Receipts and proof paths remain inspectable. AE See-Suite should still be useful without internet when local data and local tools are enough.
