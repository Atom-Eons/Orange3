# Orangebox Creations Output Stack - 2026-05-28

Orangebox Creations is a governed software factory doctrine, not a store and not one app.

AECode Source is canonical. React, Flutter, Slint, ImGui/Gio, Wails/Tauri, docs, tests, screenshots, and deployment packages are outputs.

## Active Boundary

Orangebox Ops backend may register, validate, and receipt this stack.

Orangebox Ops backend must not touch:

- website files
- visual surface files
- app factory output generation
- store-like surfaces
- production deploy execution

Those output lanes remain gated until explicitly authorized for a separate build.

## Proof

The machine-readable registry is generated by:

```powershell
npm.cmd run creations:doctor
```

The latest registry is written to:

```text
C:\Users\a\OrangeBox-Data\creations\latest-output-stack.json
```

This preserves the output doctrine without crossing the current Orangebox Ops backend boundary.
