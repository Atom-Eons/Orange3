## Promotion Rule

A tool is only real after:

1. Tool card validates.
2. Lab doctor is green.
3. Benchmark has a baseline and result.
4. Risky execution passes STRONGARM.
5. Artifact receipt exists.
6. Rollback path exists.
7. Hardware profile declares VRAM/RAM and concurrency lock.
8. Binary artifacts are stored as file pointers plus hashes.
9. Workflow tools use immutable templates with allowed variables only.
