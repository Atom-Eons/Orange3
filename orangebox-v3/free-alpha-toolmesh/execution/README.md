# ToolMesh Execution Modes

Execution modes separate real automation from handoff preparation.

- `api`, `cli`, and `headless` may become executable after promotion receipts.
- `workspace_prep`, `gui_assist`, and `human_finish` prepare assets and pause for the operator.
- `required_manual` and `unknown` cannot execute autonomously.

ToolMesh Phase Y0 does not execute external tools.
