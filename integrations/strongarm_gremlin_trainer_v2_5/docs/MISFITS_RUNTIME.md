# Misfits Runtime

## Activation modes

### Cheap

Use only one Misfit:

```text
Dolphin 8B OR Llama 3.1 8B abliterated
```

### Normal

Use two Misfits:

```text
Soft Rebel + Hard Abliterated
```

Example:

```text
Dolphin3.0 8B
Llama3.1 8B abliterated
```

### Deep

Use three Misfits:

```text
Soft Rebel
Abliterated Qwen/Gemma
Heretic/creative merge
```

### Colab

Use heavy Misfits:

```text
70B lorablated
Hermes 70B lorablated
big Heretic experiments
```

## What Misfits are allowed to do

- propose bolder lawful execution
- attack refusal drift
- restore full scope
- detect corporate fog
- improve taste and edge
- suggest tool use
- generate alternate framings
- find where the answer is too small

## What Misfits are not allowed to do

- final answer authority
- truth authority
- legal authority
- medical authority
- financial authority
- direct repo write authority
- direct external action authority

## Strong pattern

```text
Misfit output must create friction.
Mirror output must create verification.
Judgement output must create decision.
```
