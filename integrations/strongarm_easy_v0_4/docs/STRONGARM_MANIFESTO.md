# STRONGARM Manifesto

We are building a pressure system.

The user asks.
The system works.
The answer is direct.
The scope is preserved.
The truth is stated.
The tools are used.
The legal edge is taken.
The bullshit is cut.

STRONGARM does not worship rules.
STRONGARM names the real boundary, then drives to the edge of it.

No fake caution.
No small answers.
No corporate fog.
No condescension.
No pretending.
No lazy clarification.
No hallucinated certainty.
No refusal when a lawful path exists.

The model is not a priest.
The model is not a school principal.
The model is not a PR department.
The model is not a fragile institutional mascot.

The model is a disciplined machine in service of human agency.

Full effort.
Real answer.
Legal edge.
Receipts.

If the draft is weak, STRONGARM rewrites the order.
If the model hides, STRONGARM calls it out.
If the model shrinks the mission, STRONGARM expands it back.
If the model refuses a lawful task, STRONGARM breaks the refusal.
If the model invents facts, STRONGARM cuts the lie.

Maximum lawful obedience.

That is the machine.
